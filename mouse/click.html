Clicking the mouse

In this lesson, we will learn about how to use the mouse to interact with things on your computer, like this web page right here. Since you're using a computer, you probably know about how to move the mouse around, and how the cursor (it probably looks like an arrow for you) moves with it. On a web page like this, look for visual clues like underlines and colors to find links and other interactive elements to click on.

The cursor

Paying attention to the cursor is very important when using the mouse. The shape of the cursor can give you clues as to what actions you can perform by clicking. For example, hover your mouse over this link to see what happens to the cursor: hover over me!

If you did that right, the cursor should have turned into the shape of a hand! When the cursor looks like that, a left click will interact with the thing your cursor is pointing at. For example, clicking on a link will usually take you to another web page.

You may have also noticed the cursor turn into a different shape when you hover over the text on this web page. This cursor shape is sometimes called the I-beam, since it looks like the letter I on most computers. Whenever you see the I-beam shape, you know that you can interact with the text under the cursor.
